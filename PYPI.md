# NEAR JSON-RPC Client for Python

This is a Python client for interacting with the NEAR Protocol using JSON-RPC. The client allows developers to query and interact with the NEAR blockchain seamlessly.

## Features

- **Support for NEAR JSON-RPC endpoints**: Allows easy communication with NEAR blockchain nodes.
- **Query Methods**: Retrieve blockchain data, transaction status, account details, and more.
- **Asynchronous Support**: Built with `asyncio` for efficient and non-blocking operations.
- **Extensible**: Easily extendable to add more RPC methods or customize existing ones.

## Installation

You can install the client using pip:

```bash
pip install near-jsonrpc-client
